# Kotlin WebView 아카이브

Android **WebView 하이브리드 셸**의 설계·진화 패턴을, **학습·공유 목적으로 독립 재구성**한 시리즈입니다.

별도 예제 프로젝트 없이, 각 글 본문에 **Kotlin·XML 스니펫**을 포함합니다.

> 비공개 프로젝트 소스는 포함하지 않으며, URL·광고 ID·내부 버전명은 사용하지 않습니다.

> **면책** — 학습·설계 패턴 공유 목적입니다. 예제는 데모 URL·Google 테스트 AdMob ID만 사용합니다. **본인이 관리·배포 권한이 있는 앱**에만 적용하고, Play 정책·AdMob·Firebase 약관을 준수하세요.

## 작성 원칙

- 비공개 코드·URL·도메인·AdMob/Firebase 실 ID 미사용
- 기술 개념·설계 선택 중심
- 코드는 본문 인라인 스니펫 (Android Studio에 붙여 넣기)
- 데모 URL: `https://example.com` 등 placeholder만 사용
- 포스트 제목은 `Kotlin WebView — …` 접두로 통일
